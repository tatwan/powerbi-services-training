# Power BI Service Training

A comprehensive 2-day hands-on course for learning Power BI Service within the Microsoft Fabric ecosystem.

## Course Overview

This training covers essential Power BI Service skills, from basic data visualization to advanced data modeling and transformation techniques.

### Day 1: Fundamentals 
- Understanding Power BI within Microsoft Fabric
- Navigating the Power BI Service interface
- Creating semantic models and basic visualizations
- Data transformation with Power Query
- Building relationships between tables
- Creating interactive dashboards

### Day 2: Advanced Topics *(Coming Soon)*
- Advanced DAX formulas and measures
- Complex data modeling
- Performance optimization
- Sharing and collaboration
- Best practices and governance

## What You'll Build

Throughout the course, you'll work with real-world datasets including:
- **NY Property Sales**: Introduction to basic visualizations (column, bar, line charts)
- **E-commerce Sales Data**: Multi-table data model with sales, products, and store information
- **Interactive Dashboards**: Revenue analysis, product performance, and geographic insights

## Prerequisites

- Active Microsoft 365 account with Power BI access
- Basic understanding of data concepts
- No prior Power BI experience required

## Key Learning Outcomes

By the end of this course, you will be able to:
- Navigate Power BI Service and manage workspaces
- Create and publish semantic models
- Transform and clean data using Power Query
- Build relationships and data models
- Design interactive reports and dashboards
- Share insights with stakeholders
